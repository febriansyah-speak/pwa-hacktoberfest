# Apa Itu PWA?

PWA adalah website yang seakan menjadi aplikasi native

---

Aplikasi Web Progresif adalah aplikasi web yang menggunakan API dan fitur browser web yang muncul bersama dengan strategi peningkatan progresif tradisional untuk menghadirkan pengalaman pengguna seperti aplikasi asli ke aplikasi web lintas platform.

###### https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps

# Service Worker

ini yang bikin web bisa diakses secara offline
seperti yang saya bilang di slide sebelum nya

---

Mereka dimaksudkan, antara lain, untuk mengaktifkan pembuatan pengalaman offline yang efektif, mencegat permintaan jaringan dan mengambil tindakan yang sesuai berdasarkan ketersediaan jaringan, dan memperbarui aset yang berada di server.

###### https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API

```js subtitle='sw.js'
var cacheName = 'febriansyah-pwa';
var filesToCache = [
  '/',
  '/index.html'
  // masukkan semua file yang dibuat offline disini
];
self.addEventListener('install', function(e) {
  console.log('[ServiceWorker] Install');
  e.waitUntil(
    caches.open(cacheName).then(function(cache) {
      console.log('[ServiceWorker] Caching app shell');
      return cache.addAll(filesToCache);
    })
  );
});
self.addEventListener('activate',  event => {
  event.waitUntil(self.clients.claim());
});
self.addEventListener('fetch', event => {
  event.respondWith(
    caches.match(event.request, {ignoreSearch:true}).then(response => {
      return response || fetch(event.request);
    })
  );
});
```
